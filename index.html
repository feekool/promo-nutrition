📦 Установка за 1 минуту

Технические шаги одинаковы для всех — далее вы адаптируете форум под нутрициологию

  1. 1
    Зарегистрируйтесь на GitHub Создайте бесплатный аккаунт на github.com
  2. 2
    Откройте шаблон
    github.com/feekool/feekool.github.io
  3. 3
    Нажмите "Use this template" → "Create a new repository". Назовите его, например, "nutri-forum"
  4. 4
    Включите GitHub Pages: Settings → Pages → ветка main → Save. Через минуту форум онлайн!
  5. 5
    Адаптируйте под нутрициологию 🥗 Замените тексты, добавьте разделы «Курсы», «Витрина», «Отзывы» — и готово!
💡 Совет для нутрициолога: Начните с демо, протестируйте структуру, затем кастомизируйте под свою нишу: детокс, спортпит, семейное питание и др.
